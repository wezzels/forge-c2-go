# Wezzel Dashboard

## Overview

The Wezzel Dashboard (formerly DoD Dashboard) is a seismic data monitoring interface.

## Access

**URL:** http://10.0.0.117:31845/

## Features

- Station monitoring
- Event visualization
- Signal detection display
- Real-time data from GMS Simulator

## Ports

| Service | Port | Purpose |
|---------|------|---------|
| Wezzel Dashboard | 31845 | Web UI |
| GMS Simulator | 31595 | API backend |
| DoD Simulator | 30717 | Alternative UI |
